# AI Prompt Builder + Result QA

Інструмент для швидкої генерації якісних промптів та перевірки результатів видачі ШІ. Проєкт створено як односторінковий додаток (SPA), що працює локально в браузері.

Відкрити Live Demo: https://alexgreenrgb.github.io/AI-Prompt-Builder/

## Основні модулі

### 1. Конструктор промптів
Дозволяє зібрати запит на основі ключових параметрів:
- **Роль**: Задає контекстну експертність (Developer, HR тощо).
- **Тип задачі**: Визначає основну дію (Summarize, Refactor, Explain).
- **Контекст**: Обов'язкове поле для опису суті завдання.
- **Обмеження**: Тон, мова та ліміти.
- **Формат**: Вибір структури відповіді (JSON, Таблиця, Списки).

### 2. Панель керування
- **Генерація**: Автоматичне збирання тексту з полів.
- **Копіювання**: Швидкий переніс промпту в буфер обміну.
- **Очищення**: Скидання всіх полів для нової задачі.

### 3. QA Checklist & Scoring
Система оцінки відповіді ШІ за 8 критичними метриками. Користувач відмічає виконані пункти, а система:
- Рахує прогрес у форматі "X/8 пройдено".
- Змінює колір бейджа (червоний/жовтий/зелений) залежно від якості.
- **Suggest Improvements**: Динамічно генерує поради на основі невідмічених пунктів чекліста.

## Технічний стек
- **HTML5 / CSS3**: Семантична верстка та кастомний Dark Mode дизайн.
- **Bootstrap 5**: Адаптивна сітка та компоненти (cards, badges, buttons).
- **JavaScript**: Логіка обробки даних без зовнішніх залежностей.

## Інструкція із запуску
1. Розмістіть файли `index.html`, `style.css` та `script.js` в одній директорії.
2. Відкрийте `index.html` у будь-якому сучасному браузері.
3. Для коректного відображення шрифтів та іконок необхідний доступ до мережі (CDN).

## Логіка валідації
Кнопка "Generate prompt" залишається неактивною або підсвічує помилку, якщо поле **Context** порожнє. Це гарантує, що користувач не згенерує пустий запит.
